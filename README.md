# bo2McWeb

**bo2McWeb** 是為 [bo2 Minecraft 社群](https://BO2MC.bo2.tw) 所打造的網站前端系統，用於展示玩家資訊、活動公告、伺服器狀態等。

## 🔧 專案架構

本專案為網站前端，搭配後端 API 與 Minecraft 伺服器資料同步系統，提供如下功能：

- 玩家資訊展示（上線時間、積分、成就）
- 活動公告與伺服器狀態通知
- Discord 社群連結整合
- 登入與身份驗證系統（開發中）

## 📁 專案結構

bo2McWeb/
├── frontend/ # 網站前端原始碼（HTML / React / Vue）

├── README.md
